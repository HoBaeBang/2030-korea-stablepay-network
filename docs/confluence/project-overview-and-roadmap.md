# 프로젝트 개요와 로드맵

Confluence 문서: [프로젝트 개요와 로드맵](https://aslan0.atlassian.net/wiki/spaces/SPN/pages/4948097)

이 페이지는 `2030 KOREA StablePay Network`의 전체 방향, 포트폴리오 범위, 목표 아키텍처, Phase 2 구현 로드맵을 모아두는 문서 허브입니다.

Jira 티켓 번호는 작업 추적을 위한 정보이고, Confluence 문서 제목은 문서의 역할이 먼저 보이도록 관리합니다.

## 포함 문서

| 문서 | 목적 |
| --- | --- |
| [포트폴리오 프로젝트 범위](../portfolio/project-scope.md) | 이 프로젝트가 채용/학습/포트폴리오 관점에서 무엇을 보여주는지 정리합니다. |
| [목표 아키텍처](../architecture/target-architecture.md) | 현재 Phase 1 구조와 Phase 2, Phase 3 확장 방향을 설명합니다. |
| [Phase 2 구현 로드맵](../roadmap/phase-2-roadmap.md) | Ledger, Settlement, Indexer, Deposit, Withdrawal, Wallet, Rust 확장 순서를 정리합니다. |

## 운영 원칙

- 문서 제목은 `어떤 내용을 설명하는가`를 기준으로 정합니다.
- Jira 티켓 번호는 본문 상단의 관련 Jira 항목에만 남깁니다.
- GitHub 원본 Markdown과 Confluence 학습 문서는 서로 연결해 둡니다.
