# Phase 2 도메인 학습

Confluence 문서: [Phase 2 도메인 학습](https://aslan0.atlassian.net/wiki/spaces/SPN/pages/4948117)

이 페이지는 Phase 2에서 필요한 블록체인 결제/금융 백엔드 도메인 지식을 모아두는 학습 허브입니다.

Phase 2의 핵심은 단순히 API를 추가하는 것이 아니라, Payment 상태를 Ledger, Settlement, Blockchain Event Indexer, Deposit, Withdrawal, Wallet, Key Security와 연결해 실제 블록체인 금융 백엔드에 가까운 구조를 이해하는 것입니다.

## 학습 순서

| 순서 | 문서 | 목적 |
| --- | --- | --- |
| 1 | [Phase 2 전체 도메인 지도](../domain/phase-2-domain-map.md) | Phase 1과 Phase 2의 차이, 전체 도메인 흐름을 이해합니다. |
| 2 | [Ledger와 Settlement 기초 학습](../domain/ledger-and-settlement-basic-learning.md) | Payment, Ledger, Settlement의 책임을 분리해서 이해합니다. |
| 3 | Deposit, Withdrawal, Wallet, Key Security | 입출금과 지갑/키 보안 경계를 학습합니다. |
| 4 | Blockchain Event Indexer와 운영 개념 | 온체인 이벤트 감지, 멱등성, 대사, 재처리를 학습합니다. |

## 운영 원칙

- 출퇴근 시간에는 개념 학습 문서를 읽습니다.
- 퇴근 후에는 GitHub repo에 직접 도메인 문서를 작성합니다.
- 검증문제는 먼저 풀고, 답변가이드는 나중에 확인합니다.
- 모르는 용어는 용어사전 또는 다음 학습 문서에 계속 보강합니다.
