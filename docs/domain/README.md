## 폴더별 역할

### 00_종합

Day1~Day5 전체를 반복 복습하거나, Day6 통합 점검을 진행할 때 보는 문서입니다.

### 01_Phase2_전체지도

Phase 1 결제 백엔드가 Phase 2에서 어떤 블록체인 금융 백엔드로 확장되는지 큰 그림을 잡는 문서입니다.

### 02_원장과_정산

Payment, Ledger, Settlement의 책임 차이와 돈의 이동 기록, 정산 개념을 학습합니다.

### 03_입출금과_지갑보안

Deposit, Withdrawal, Wallet, Key Security, Rust Signer 경계를 학습합니다.

### 04_블록체인_이벤트_인덱서

Blockchain Event Indexer, Polling, Checkpoint, Idempotency, Reconciliation을 학습합니다.

### 05_구현범위와_Sprint2

Phase 2에서 무엇부터 구현할지, Sprint 2 Backend Core를 왜 먼저 정리하는지 학습합니다.

### 06_백엔드코어

Sprint 2 구현을 시작하기 전에 공통 에러 응답, 요청 검증, 설정, 로그, 테스트 패턴을 정리합니다.
